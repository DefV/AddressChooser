Constructor

 new Mapeed.Proxy.GoogleMap(element, callback, context)

Creates a new Mapeed.Proxy.GoogleMap object used by Mapeed.AddressChooser.Plugin. Calls calling on context when map is initiliazed and ready to use By default map displays the world but it can be changed when callback is called
