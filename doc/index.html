Introduction

Mapeed.AddressChooser is a Javascript script to create a nice address form for any websites that need to collect addressed. For example for a user, for a place or for anything that can be localized on a map.

This widget is mainly inspired by Google Local Business Center.

Features

Mapeed.AddressChooser's features are:
  • No framework dependent. You can use it as is or with any great frameworks like Prototype, JQuery, YUI ...
  • No mapping system dependent. The current implementation is based on Google Map using a Google Map proxy object.
    But you can create your own map proxy (Yahoo Map for instance). You need to implement public method of Map Proxy object.
  • Interactive map display location while you are typing an address.
  • Center map on user location (based on its IP) if mapping system allows it.
  • Works on Safari, Firefox, IE 6/7, Chrome and Opera
  • Documentation generated with PDoc
  • Packed version under 6Ko
  • Open source, code available on github
  • ...
