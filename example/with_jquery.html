Mapeed.AddressChooser / JQuery example

This example uses Mapeed.AddressChooser with JQuery framework. Address is an autocomplete field filled by addresses returns by Google Map:

  • Default options
  • Callback on onInitialized to add maps controls
  • Callback on onSuggestsSearch to display spinner on address field
  • Callback on onSuggestsFound to remove spinner on address field and fill autocomplete

Address string is sent to Google while you are typing (in fact you stop typing for 300ms) and results are display automatically in the autocomplete field.
